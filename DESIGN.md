# Customer Segmentation Dashboard - Design Document

## Design Philosophy: Data-Driven Elegance

This dashboard embodies a modern, professional aesthetic focused on clarity and actionable insights. The design prioritizes data visualization with a sophisticated color palette, deliberate typography hierarchy, and smooth micro-interactions.

### Color Palette
- **Primary**: Deep Blue (#1e40af) - Trust, data-focused
- **Secondary**: Slate Gray (#475569) - Professional, neutral
- **Accent**: Emerald Green (#059669) - Positive insights, growth
- **Warning**: Amber (#d97706) - At-risk segments
- **Danger**: Rose (#e11d48) - Churn risk
- **Background**: Clean White (#ffffff) with subtle gray accents
- **Text**: Dark Slate (#1e293b) for readability

### Typography
- **Display**: Geist (Bold, 32-40px) - Page titles
- **Heading**: Geist (Semi-bold, 20-24px) - Section headers
- **Body**: Inter (Regular, 14-16px) - Content and labels
- **Mono**: JetBrains Mono (Regular, 12-14px) - Code snippets

### Layout Structure
- **Hero Section**: Full-width gradient background with key metrics
- **Dashboard Grid**: 3-column layout for segment cards
- **Visualization Area**: Large, prominent chart displays
- **Code Section**: Tabbed interface with syntax highlighting
- **Footer**: Minimal, clean design with links

### Micro-interactions
- Hover effects on cards (subtle lift, shadow enhancement)
- Smooth transitions (200-300ms) for all state changes
- Loading states with skeleton screens
- Tooltip reveals on data points
- Smooth scroll behavior

## Key Sections

1. **Header**: Navigation and branding
2. **Hero**: Project overview and key statistics
3. **Segments**: Interactive segment cards with RFM metrics
4. **Visualizations**: 4 key charts showcasing analysis
5. **Insights**: Business recommendations
6. **Code**: Embedded code snippets with syntax highlighting
7. **Footer**: Links and attribution
